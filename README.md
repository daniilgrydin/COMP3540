# COMP3540

Assignments for COMP3540 Advanced Web Design and Programming
Everything is published on the [TRU website](https://cs.tru.ca/~T00712793/)

- `WeekX` - contains assignments from that respective week of classes.
- `Utils/assignment.php` - script that can be imported to add a stylized view to assignments.
- `permit.py` - updates the file and folder permissions so that the files are viewable on the web-platform.
